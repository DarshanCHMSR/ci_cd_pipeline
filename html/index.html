Project Overview

This project demonstrates a complete AWS-focused CI/CD setup that automates application build, validation, image creation, and deployment. The goal is to reduce manual release steps and provide predictable, repeatable production updates for a containerized web app.

Technologies Used

  • AWS ECS
  • AWS ECR
  • AWS CodeCommit
  • AWS CodeBuild
  • AWS CodeDeploy
  • Docker

CI/CD Workflow

  1. Code is pushed to AWS CodeCommit.
  2. CodeBuild validates and builds the application image.
  3. The Docker image is published to Amazon ECR.
  4. CodeDeploy rolls the update to containers running on ECS.
